# Contributing

Contributions are welcome — bug fixes, new widgets, themes, or any improvements.

## Getting started

```bash
git clone https://github.com/yasir-ali9/claude-code-line.git
cd claude-code-line
npm install
npm run build
node dist/index.js
```

## Making changes

1. Fork the repo and create a branch
2. Make your changes
3. Run `npm run build` and test locally
4. Open a pull request with a short description of what you changed and why

## Ideas

- New widgets
- New themes
- Better defaults
- Bug fixes
- Documentation improvements

No contribution is too small.
